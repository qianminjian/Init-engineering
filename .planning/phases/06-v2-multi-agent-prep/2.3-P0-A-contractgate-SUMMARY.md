# Phase 06 Plan 2.3-P0-A Summary: ContractGate 真实实现 (不再 placeholder)

## Goal

将 ContractGate 从永远的 `passed("skip"/"placeholder")` 升级为实际检查 `.ae-contracts/` 目录下的契约文件存在性和格式正确性.

## Problem

`auto_engineering/gates/contract.py` 第 55-63, 76-77 行 — 多 Agent 场景永远返回 `passed("skip: multi-agent but no contracts defined")` 或 `passed("placeholder")`. 7 道 Gate 中 1 道是假的.

## Solution

1. **单 Agent 模式** — 保持 skip (passed=True), 语义不变
2. **多 Agent + 显式 contracts dict** — 校验类型后 passed (保留参数接口)
3. **多 Agent + 磁盘契约文件** — 检查 `.ae-contracts/` 目录存在性 → 收集 `.yml/.yaml/.json` 文件 → 逐文件 YAML/JSON parse → 全部成功才 passed
