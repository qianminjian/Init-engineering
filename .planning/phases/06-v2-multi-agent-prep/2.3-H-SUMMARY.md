## Decisions Made

- **AgentRuntime.get(role) vs _get_or_create_agent(agent_type):** 暴露 public `get(role)` API (返回 None vs 抛 LookupError), 让 Orchestrator._build_runtime_executor 优雅处理"未注册"场景 (返回 failed TaskOutcome 而非 crash).
- **未注册 role 处理:** 借鉴 Plan Mode 三问协议, 单 task 失败不导致整个 round 失败. 返回 `TaskOutcome(status='failed', error="No agent registered for role: {role}")` 让 Round 继续.
- **mock architect/critic:** Phase H 是"集成"任务, 不是"实现 agent". v2 demo 中 architect/critic 用 _MockRoleAgent 简化, 避免引入额外 LLM 调用风险. 真 LLM 测试在后续 Phase (或单独 task).
- **DeveloperAgent adapter:** DeveloperAgent 已有 `execute(task, ctx, ...)` 协议, 但需要包成 `_DeveloperAgentAdapter` 记录 execute_calls (进度日志) — 避免修改 DeveloperAgent.
- **保留 _run_gates / _evaluate_semantic:** Phase G P1.3 决策不变, 向后兼容 (外部代码可能 import).
